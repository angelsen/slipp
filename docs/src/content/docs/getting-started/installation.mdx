## Requirements

- Python 3.12+
- [uv](https://docs.astral.sh/uv/) (recommended) or pip
- Ansible (for deployments)

## Quick Install

<Steps>

1. **Install slipp**

   <Tabs syncKey="pkg">
   <TabItem label="uv (recommended)">
   ```bash
   uv tool install slipp
   ```
   </TabItem>
   <TabItem label="pip">
   ```bash
   pip install slipp
   ```
   </TabItem>
   </Tabs>

2. **Verify installation**

   ```bash
   slipp --version
   ```

3. **Enable shell completion** (optional)

   <Tabs syncKey="shell">
   <TabItem label="Bash">
   ```bash
   slipp --install-completion bash
   source ~/.bashrc
   ```
   </TabItem>
   <TabItem label="Zsh">
   ```bash
   slipp --install-completion zsh
   source ~/.zshrc
   ```
   </TabItem>
   <TabItem label="Fish">
   ```bash
   slipp --install-completion fish
   ```
   </TabItem>
   </Tabs>

</Steps>

## Development Setup

For contributing or local development:

<Steps>

1. **Clone the repository**

   ```bash
   git clone https://github.com/angelsen/slipp.git
   cd slipp
   ```

2. **Install dependencies**

   ```bash
   uv sync
   ```

3. **Run slipp**

   ```bash
   uv run slipp --help
   ```

</Steps>

:::tip
Install Ansible separately if you plan to deploy:
```bash
uv tool install ansible
```
:::

## Provisioning a VPS with Gigahost

You don't need to bring your own VPS — slipp can order and bootstrap one
for you via [Gigahost](https://gigahost.no):

<Steps>

1. **Add the provider**

   ```bash
   slipp providers add gigahost
   ```

   Prompts for a Gigahost API key and verifies it.

2. **Provision and deploy in one command**

   ```bash
   slipp up myapp --domain myapp.example.com
   ```

   Orders a VPS, bootstraps the `slipp` service account, registers (or
   registers-and-syncs) the domain, generates the deployment with `slipp
   launch`, converges DNS, and runs `slipp deploy` — end to end.

   Or do it in stages: `slipp provision myapp` (order + bootstrap only),
   then `slipp launch`/`slipp deploy` as usual.

</Steps>

See [`slipp up`](/reference/cli#up), [`slipp provision`](/reference/cli#provision),
and [`slipp server install`](/reference/cli#server) (OS reinstall + resumable
bootstrap) for the full command set.
