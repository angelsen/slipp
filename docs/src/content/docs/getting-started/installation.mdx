## Requirements

- Python 3.12+
- [uv](https://docs.astral.sh/uv/) (recommended) or pip

## Install slipp

<Tabs syncKey="pkg">
  <TabItem label="uv">
    ```bash
    uv tool install slipp
    ```
  </TabItem>
  <TabItem label="pip">
    ```bash
    pip install slipp
    ```
  </TabItem>
</Tabs>

Or install from GitHub (latest):

<Tabs syncKey="pkg">
  <TabItem label="uv">
    ```bash
    uv tool install git+https://github.com/angelsen/slipp
    ```
  </TabItem>
  <TabItem label="pip">
    ```bash
    pip install git+https://github.com/angelsen/slipp
    ```
  </TabItem>
</Tabs>

## For Development

```bash
git clone https://github.com/angelsen/slipp.git
cd slipp
uv sync
uv run slipp --help
```
