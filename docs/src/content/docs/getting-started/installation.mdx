## Requirements

- Python 3.12+
- [uv](https://docs.astral.sh/uv/) (recommended) or pip
- Ansible (for deployments)

## Quick Install

<Steps>

1. **Install slipp**

   <Tabs syncKey="pkg">
   <TabItem label="uv (recommended)">
   ```bash
   uv tool install slipp
   ```
   </TabItem>
   <TabItem label="pip">
   ```bash
   pip install slipp
   ```
   </TabItem>
   </Tabs>

2. **Verify installation**

   ```bash
   slipp --version
   ```

3. **Enable shell completion** (optional)

   <Tabs syncKey="shell">
   <TabItem label="Bash">
   ```bash
   slipp --install-completion bash
   source ~/.bashrc
   ```
   </TabItem>
   <TabItem label="Zsh">
   ```bash
   slipp --install-completion zsh
   source ~/.zshrc
   ```
   </TabItem>
   <TabItem label="Fish">
   ```bash
   slipp --install-completion fish
   ```
   </TabItem>
   </Tabs>

</Steps>

## Development Setup

For contributing or local development:

<Steps>

1. **Clone the repository**

   ```bash
   git clone https://github.com/angelsen/slipp.git
   cd slipp
   ```

2. **Install dependencies**

   ```bash
   uv sync
   ```

3. **Run slipp**

   ```bash
   uv run slipp --help
   ```

</Steps>

:::tip
Install Ansible separately if you plan to deploy:
```bash
uv tool install ansible
```
:::
