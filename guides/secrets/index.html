Troubleshooting

Section titled “Troubleshooting”

Browser didn’t open?

Copy the URL from terminal output and open manually.

Timeout expired?

Increase timeout: slipp secrets pull nor-auth --timeout 600

Connection refused?

Ensure no firewall blocks localhost callback port (49152-65535 range).

Troubleshooting

Section titled “Troubleshooting”

Vault password incorrect?

Check you’re using the correct password. Re-encrypt if needed:

Secret not found?

  • Verify vault path: slipp secrets list
  • Check secret name matches: slipp secrets list myproject
  • Ensure vault is encrypted: file vault.yml should show “ASCII text”
